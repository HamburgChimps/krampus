# 2022

Solutions to the [Advent of Code 2022](https://adventofcode.com/2022) in Java using Quarkus.

Install [quarkus](https://quarkus.io/get-started/) and then run `quarkus dev` in this directory.

Afterwards visit (make A GET request to) <http://localhost:8080/day/:number/part/:number> to execute a specific solution.

For example, to execute the solution for part 2 of day 5, you would make a GET to <http://localhost:8080/day/5/part/2>.
